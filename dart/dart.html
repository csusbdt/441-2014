Dart Assignment

Learning Objectives

  • Learn how to use the Dart Editor to create client-side Web applications.

Preparation

To complete this assignment, you need to have an account that enables you to log into the computers in JB 360 or you need to have a laptop that is capable of running the Dart Editor.

If you are in JB 360, then the Dart editor is already installed. If you are using your own laptop or desktop computer, then you should download and install the current stable release of Dart + Editor

Instructions

  • Become familiar with the Dart website.
  • Watch Dart for the modern web developer.
  • Complete the Get Started Tutorial.
  • Read Chapter 1: Quick Start.
  • Study Chapter 2. A Tour of the Dart Language.
  • Become familiar with the Event Loop and Dart.

  print('red');

main() {
  print('red');
  Future future = new Future(() => print('blue'));
  Future future2 = future.then((_) => throw new Exception());
  future2.catchError((e) => print('orange'));
  print('green');
}
