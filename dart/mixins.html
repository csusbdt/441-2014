Instructions

Create a dart project named ui by making a copy of the point-light folder.

We will add a button that is used to pause the animation.

Create a file ui.dart with the following. 

library ui;

import 'dart:html';

bool isPaused = false;

void init() {
  Element pauseButton = document.createElement('button')
      ..innerHtml = 'Pause';
  
  document.body.append(pauseButton);
    
  pauseButton.onClick.listen((_) {
    isPaused = !isPaused;
  });
}

Add code to main to call the init method of the ui library.

Add code to the scene library to read the state of isPaused in the ui library and pause or resume the animation accordingly. Make sure the animation pauses and resumes smoothly, without suddenly jumping to a new position.
