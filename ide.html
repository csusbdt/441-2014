IDE

Overview

In this class you should be able to work in either Visual Studio or Xcode (or an IDE for Linux if you are running Linux). I will demonstrate both of these IDEs in this course.

Install

If you are working in JB 360, Visual Studio 2010 is already installed, so you don't need to install anything.

There is a free version of Visual Studio that you can use for this course called Visual C++ 2010 Express.

Xcode is a free download on the App Store.
